## Créer un compte

1. Rendez-vous sur la page **S'inscrire** (`/signup`).
2. Remplissez votre **nom**, **email** et **mot de passe**.
3. Cliquez sur **Créer un compte**.
4. Consultez votre boîte email et cliquez sur le **lien de vérification** reçu.
5. Votre compte est activé - vous pouvez vous connecter.

## Méthodes de connexion

Roadmaps Faciles propose plusieurs méthodes d'authentification.

### Connexion par mot de passe

C'est la méthode principale :

1. Rendez-vous sur la page de connexion (`/login`).
2. Entrez votre **email** et **mot de passe**.
3. Cliquez sur **Se connecter**.

Si vous avez oublié votre mot de passe, cliquez sur **Mot de passe oublié ?** pour recevoir un lien de réinitialisation par email.

### Connexion sans mot de passe (magic link)

Si vous préférez ne pas utiliser de mot de passe :

1. Sur la page de connexion, cliquez sur **Se connecter sans mot de passe**.
2. Entrez votre **adresse email**.
3. Consultez votre boîte email et cliquez sur le **lien de connexion** reçu.

Le lien est valable pour une seule utilisation.

### Connexion via Espace Membre (beta.gouv.fr)

Si vous êtes membre de l'écosystème beta.gouv.fr :

1. Sur la page de connexion, cliquez sur **Connexion Espace Membre**.
2. Entrez votre **identifiant Espace Membre** (ex. : `prenom.nom`).
3. Consultez votre email et cliquez sur le lien reçu.

Une fois connecté, vous pouvez accéder à n'importe quel espace de travail grâce au **pont SSO** : votre session est automatiquement transférée lorsque vous naviguez vers un sous-domaine.

### Connexion via un fournisseur OAuth (SSO)

Si l'administrateur de l'espace a activé des fournisseurs OAuth, vous pouvez vous connecter en un clic :

1. Rendez-vous sur la page de connexion de l'espace.
2. Cliquez sur le bouton du fournisseur souhaité (**GitHub**, **Google** ou **ProConnect**).
3. Autorisez l'accès sur la page du fournisseur.
4. Vous êtes automatiquement connecté et rattaché à l'espace.

Les fournisseurs disponibles dépendent de la configuration de l'espace. Si aucun bouton OAuth n'apparaît, cette option n'est pas activée.

### Connexion via invitation

Si vous recevez un email d'invitation :

1. Cliquez sur le **lien d'invitation** dans l'email.
2. Vous arrivez sur la page de connexion de l'espace avec l'invitation pré-chargée.
3. Connectez-vous normalement (email ou Espace Membre).
4. Votre compte est automatiquement rattaché à l'espace avec le rôle défini par l'administrateur.

## Inscription et politique d'accès

Chaque espace de travail définit sa propre politique d'inscription :

- **Tout le monde** - N'importe quelle adresse email peut créer un compte.
- **Domaines autorisés** - Seules les adresses email d'un domaine spécifique sont acceptées (ex. : `@mon-organisation.fr`).
- **Sur invitation uniquement** - L'inscription est fermée, seuls les membres invités peuvent rejoindre l'espace.

Si vous n'arrivez pas à vous inscrire, vérifiez la politique d'accès avec l'administrateur de l'espace.

## Appartenir à plusieurs espaces

Un même compte (identifié par votre email) peut être membre de **plusieurs espaces de travail**. Vous pouvez avoir des rôles différents sur chaque espace.

## Double authentification (2FA)

Certains espaces peuvent exiger la **double authentification**. Dans ce cas, après votre première connexion, vous serez invité à configurer une méthode 2FA (clé d'accès, application OTP ou email) avant de pouvoir accéder à l'espace.

Vous pouvez gérer vos méthodes 2FA depuis votre **Profil > Sécurité**. Voir [Authentification](/doc/admin/authentication) pour plus de détails.
